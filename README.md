# vant-element-examples

vant-element 组件的演示项目，基于 Vue CLI (Vue 2)。

## 使用

```bash
# 安装依赖
npm install

# 本地开发（自动使用本地 vant-element）
npm run dev

# 构建
npm run build
```

## 项目结构

```
src/
├── main.js            # 入口，注册 vant-element
├── App.vue            # 根组件
├── router/            # 路由配置
└── views/             # 每个组件对应一个演示页面
    ├── ButtonPage.vue
    ├── InputPage.vue
    └── ...
```
